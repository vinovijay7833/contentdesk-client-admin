## 📖 Overview

ContentDesk is a simple **Client & Admin** website developed using **HTML, CSS, and JavaScript**.

The project demonstrates a content writing service platform where clients can explore available services and administrators can manage service information through a simple dashboard.

---

## ✨ Features

- 🏠 Home Page
- ✍️ Content Writing Services
- 📋 Admin Dashboard
- 📱 Responsive Design
- 🎨 Clean User Interface
- ⚡ JavaScript Interactions
- 📊 Recent Orders Table
- 📞 Contact Section
